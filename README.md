# Intelligent-Sensing-Lab


## Summary 
---
A web development project for the course CS2204 Fundamentals of Internet Applications Development at CityUHK about 'Intelligent Sensing Lab' (ISL).

## Features
---
- Animations, gradients, transitions, and different positioning formats
- Visually-appealing yet minimalistic design
- Navigation bar
- Responsive design and liquid layout
- Multi-page layout
- Multi-column layout
- Efficient event-handling using JavaScript

## Description Of Pages
---
**Home page:** Introduces ISL, giving its location and a link to apply to it.
**About page:** Provides in-depth details about the highlights and achievements of ISL.
**Apply page:** Shows all the groups in the three divisions one can apply to, which, and a form where they can rank each group that they are applying to. 
**Visit page:** Has a form with appropriate error-handling where the user can enter a date and time for making a booking

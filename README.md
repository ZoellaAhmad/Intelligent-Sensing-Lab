# Intelligent-Sensing-Lab


## Summary 
A web development project for the course CS2204 Fundamentals of Internet Applications Development at CityUHK about 'Intelligent Sensing Lab' (ISL).

## Features
- Animations, gradients, transitions, and different positioning formats
- Visually-appealing yet minimalistic design
- Navigation bar
- Responsive design and liquid layout
- Multi-page layout
- Multi-column layout
- Efficient event-handling using JavaScript

## Description Of Pages
**Home page:** Introduces ISL, giving its location and a link to apply to it.<br>
**About page:** Provides in-depth details about the highlights and achievements of ISL.<br>
**Apply page:** Shows all the groups in the three divisions one can apply to, which, and a form where they can rank each group that they are applying to.<br>
**Visit page:** Has a form with appropriate error-handling where the user can enter a date, time, and number of visitors to check whether or not a booking is available.<br>
**Information page:** Further provides more information, has a banner which loops between three headings continuously, and plays two videos, one after the other, on loop. The videos may not load due to some issues.<br>
**Contact page:** Lists contact information.<br>
**Design page:** Gives a detailed description of all CSS techniques (animations, transitions, box-shadows, etc.) and JavaScript techniques (ES6 features and event-handling) used in the project, with line numbers of where they are used. This page can be accessed by clicking the link in the footer of any of the above webpages.
